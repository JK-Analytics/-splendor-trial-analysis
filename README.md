# Splendor Analytics Trial Activation Analysis

## Overview
This project analyses behavioural data from Splendor Analytics' 
30-day free trial to define Trial Activation and understand 
what drives conversion to paid customers.

## Project Structure
- `Data/` - Raw dataset
- `notebooks/` - Python analysis notebooks
- `SQL/` - SQL models for the data warehouse

## Tasks
- Task 1: Data Cleaning & EDA
- Task 2: SQL Models (Trial Goals & Trial Activation)
- Task 3: Descriptive Analytics & Product Metrics
